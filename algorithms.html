Disk Scheduling Algorithms

Understand how each algorithm works and how it manages disk requests efficiently.

FCFS (First Come First Serve)

Processes requests in the same order they arrive. It is simple but may cause high seek time.

SSTF (Shortest Seek Time First)

Selects the nearest request first. It reduces seek time but may starve far requests.

SCAN

Disk head moves in one direction servicing requests, then reverses direction like an elevator.

C-SCAN

Head moves in one direction only. After reaching the end, it jumps back to start.

LOOK

Similar to SCAN, but head only goes as far as the last request instead of disk end.

C-LOOK

Like C-SCAN, but only moves up to the last request then jumps to first request.
