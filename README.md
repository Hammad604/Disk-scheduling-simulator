# 💻 Disk Scheduling Simulator

A web-based Operating System project developed to visualize and compare different Disk Scheduling Algorithms through an interactive simulator, performance metrics, and graphical analysis.

---

## 📌 Project Overview

Disk Scheduling is an important concept in Operating Systems.  
When multiple disk requests arrive, the OS must decide the order in which requests should be served to reduce seek time and improve performance.

This project provides a user-friendly simulator where users can enter disk requests, choose an algorithm, and instantly view execution results with charts.

---

✅ Modern User Interface

✅ Easy to Use Navigation  
✅ Responsive Design

---

## 📚 Implemented Algorithms

- FCFS (First Come First Serve)
- SSTF (Shortest Seek Time First)
- SCAN
- C-SCAN
- LOOK
- C-LOOK

---

## 📊 Output Metrics

The simulator displays:

- Execution Order
- Total Seek Time
- Average Seek Time
- System Throughput
- Algorithm Type (Non-Preemptive)
- Graph Comparison
